## Task Summary

## Backend Change Summary

- Added/updated auth session v2 API behavior for web/native wallet sessions, refresh/logout, nonce/signature handling, and connection transfer/redeem flows.

- Hardened structured wallet signature verification, replay protection, allowed-domain/audience checks, and websocket authentication expiry handling.
- Added `/api/settings.auth` rollout settings so web can learn backend strict-mode and migration-deadline state without redeploying frontend.
- Updated OpenAPI/auth contract surfaces and generated API artifacts where applicable.
- Added and iterated focused auth/session/signature tests.
